终于到了安装主角的时候啦。

  • 安装完成，查看版本：hexo -v

修改配置主题

  • hexo初始化之后默认的主题是landscape , 然后你可以去这个地址里面找到你想要的主题。在github中搜索你要的主题名称，里面都会有该主题的如何使用的介绍。

写文章

  • 新建文章：hexo new '文章名'，然后你就可以在source/_posts路径下看到你创建的文章啦，编辑完成之后按照前面说的方式部署，在浏览器刷新就能看到你的文章了。
  • 关于具体的文章编辑你可以看下官网的介绍
  • 至于markdown，推荐目前在用的一款编辑器，很酷：typora
